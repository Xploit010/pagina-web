Que es un hacker

Un hacker es una persona que utiliza sus habilidades informáticas para robar, modificar, o manipular sistemas, aplicaciones, o datos de otros usuarios o sistemas. El término hacker se utilizó en la era digital en la década de 1960, y ahora se utiliza para referirse a cualquier persona que utiliza sus habilidades informáticas para robar, modificar, o manipular sistemas, aplicaciones, si quieres empezar aprender en el enlace que esta abajo de dejo un curso:


Tipos de hacker

  1. Crackers: Son hackers que utilizan la tecnología para robar contraseñas de cuentas de sistemas.
  2. Piratas informáticos: Son hackers que utilizan la tecnología para robar, modificar, o manipular sistemas, aplicaciones, o datos de otros usuarios o sistemas.
  3. Vigilantes: Son hackers que utilizan la tecnología para robar, modificar, o manipular sistemas, aplicaciones, o datos de otros usuarios o sistemas para obtener información confidencial o para proteger los sistemas.
